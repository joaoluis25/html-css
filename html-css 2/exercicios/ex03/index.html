Para o uso de negrito de usa a tag <STRONG> e a tag <B> mas a tag b não é mais utilizada.
Para se dar ênfase se usa o italico com a tag <EM> usada para dar enfase a uma fala por exemplo, tambem pode ser visto a tag <I> mas ela não e mais usada.
Se voce deseja fazer marcações em um texto se usa a tag <MARK> ex: Lorem, ipsum dolor sit amet consectetur adipisicing elit. Pariatur delectus dignissimos quisquam dolore excepturi at, repellendus, laudantium molestiae consequatur, eligendi dicta facilis fuga totam quae numquam earum veniam labore tenetur? o estilo pode ser adaptavel a sua escolha.
Para deixar textos menores ainda se usa a tag <SMALL> ex: pipa pau gosta de bater na madeira. porem a irma dela a tag BIG não e mais usada.
Para se usar textos deletados se usa a tag <DEL> frase deletada com sucesso covouski., antigamente se usava a tag STRIKE mas ela não é mais usada.
Para se inserir um texto ou seje, mostrar que ele deve ser lido se usa a tag <INS>texto de grande importancia.
Para se escrever os numeros de O2 e H2O se usa as tags <SUP> e <SUB> um para em cima e outro para baixo.
Para se usar citações se usa as tags <q> usada para citações curtas no meio de paragrafos por ex. Lorem ipsum dolor sit amet consectetur adipisicing elit. Saepe id sint corrupti nobis, dolore magnam tempore esse! Nostrum deserunt fugit natus perspiciatis explicabo reprehenderit ea. Nostrum deleniti possimus necessitatibus explicabo. Para paragrafos inteiros de citação se usa a tag <BLOCKQUOTE>

Para se usar abreviações se usa a tag <ABBR> ex: p.
E para se inverter textos se usa a tag <BDO> com as referencias RTL e LTR ex: joao luis=joao luis.
